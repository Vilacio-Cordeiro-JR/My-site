My-site - Portfólio Profissional

Visão Geral

Site pessoal desenvolvido para apresentar portfólio, currículo e projetos profissionais. Construído com tecnologias web fundamentais (HTML5, CSS3, JavaScript), o projeto demonstra habilidades em desenvolvimento front-end, design responsivo e boas práticas de versionamento com Git.

Tecnologias Utilizadas

  • Frontend: HTML5 semântico, CSS3 com variáveis customizadas, JavaScript ES6+
  • Estilização: CSS modular com arquitetura baseada em componentes
  • Versionamento: Git e GitHub para controle de versão
  • Deploy: Vercel com configuração otimizada
  • Recursos: Intersection Observer API, animações CSS, design responsivo

Funcionalidades

  • Página principal com apresentação profissional e sistema de posts
  • Currículo completo em página separada para impressão/compartilhamento
  • Seção de projetos com cards interativos e páginas de detalhes
  • Formação acadêmica com estrutura de posts estilizados
  • Cálculo automático de idade com JavaScript
  • Animações de contagem crescente nas estatísticas
  • Design 100% responsivo para mobile e desktop
  • Sistema de navegação consistente em todas as páginas
  • Estrutura modular e escalável para novos conteúdos

Objetivos

Criar um site profissional moderno para apresentação de portfólio e currículo, utilizando tecnologias web fundamentais com foco em performance, acessibilidade e design responsivo. O projeto busca demonstrar competências técnicas em desenvolvimento front-end, além de servir como plataforma para compartilhamento de projetos e experiências profissionais.
