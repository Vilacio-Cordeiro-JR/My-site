Sistema de Orçamentos RadCom

Visão Geral

Sistema completo de gestão de orçamentos com autenticação, multi-sede e relatórios de vendas. Desenvolvido para a empresa RadCom, permite gestão eficiente de orçamentos, controle de vendas por sede, sistema de permissões hierárquico e geração de relatórios visuais.

Tecnologias Utilizadas

  • Backend: Node.js 18+, Express 4.18, SQLite/PostgreSQL, Knex.js, bcryptjs
  • Frontend: React 18.2, Vite 4.4, Tailwind CSS, Chart.js, React Router DOM
  • Banco de Dados: SQLite (desenvolvimento) / PostgreSQL (produção)
  • Deploy: Railway (backend), Vercel/AWS Amplify (frontend)
  • Ferramentas: Docker, Git, Excel Integration

Funcionalidades

  • Autenticação completa: Login, registro, perfis de usuário
  • Sistema de cargos: Hierarquia com permissões (Dono, Gerente, Técnico, Vendedor, Atendente)
  • Gestão de orçamentos: Criar, editar, salvar, carregar cotações
  • Multi-sede: Vendas rastreadas por localização (Matriz, Filiais)
  • Confirmação de vendas: Com validação de desconto e aprovação ADM
  • Relatórios visuais: Gráficos de linha com histórico de vendas por técnico
  • Permissões granulares: ADM/criador podem excluir orçamentos
  • SQL Viewer: Interface para visualizar e consultar banco de dados
  • Catálogo de produtos: Importação via Excel
  • Geração de PDF: Orçamentos exportáveis

Objetivos

Desenvolver um sistema completo de gestão de orçamentos para a empresa RadCom, automatizando processos de vendas, implementando controle de acesso por hierarquia de cargos, permitindo gestão multi-sede e fornecendo relatórios visuais para análise de desempenho de vendas. O sistema busca eficiência operacional, segurança de dados e facilidade de uso para diferentes níveis de usuários.
